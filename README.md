# 图像几何变换实验说明

## 实验目的
通过实验观察并总结相似变换、仿射变换和透视变换对图像几何性质的影响，包括直线保持性、平行保持性、垂直保持性和圆保持性，并掌握使用透视变换校正透视畸变图像的方法。

## 实验环境
- Python 3.x
- OpenCV 4.13.0
- NumPy
- Matplotlib
- Tkinter（用于文件选择对话框）

## 实验内容

### 1. 测试图像生成
创建一张包含以下元素的测试图（500×500像素）：
- 红色矩形
- 绿色圆形
- 蓝色水平平行线
- 黄色垂直线

### 2. 三种变换实验

**相似变换**：使用旋转30度加缩放0.8倍，观察变换后图形属性变化。

**仿射变换**：通过三个对应点计算变换矩阵，观察变换后图形属性变化。

**透视变换**：通过四个对应点计算变换矩阵，观察变换后图形属性变化。

### 3. 透视畸变校正
- 支持从本地选择图片或使用模拟测试图片
- 在图像上手动点击四个角点（左上、右上、左下、右下）
- 计算透视变换矩阵并进行校正
- 对比显示原始畸变图像和校正后图像

## 实验结果总结

| 变换类型 | 直线 | 平行线 | 垂直线 | 圆 |
|---------|------|--------|--------|-----|
| 相似变换 | 保持直线 | 保持平行 | 保持垂直 | 保持圆 |
| 仿射变换 | 保持直线 | 保持平行 | 不保持垂直 | 变为椭圆 |
| 透视变换 | 保持直线 | 不保持平行 | 不保持垂直 | 变为椭圆或复杂曲线 |

## 实验心得

通过本次实验，我深入理解了三种几何变换的本质区别与各自特性。相似变换能够保持图形的所有几何属性，是最严格的变换；仿射变换次之，保持直线和平行性但会破坏垂直关系；透视变换仅能保持直线性。在透视畸变校正实践中，掌握了通过四点对应计算透视变换矩阵的方法。实验中也遇到了OpenCV的Qt后端兼容性问题，最终通过使用matplotlib的事件处理机制解决，提升了解决实际问题的能力。
