# CNN 手写数字识别实验说明（PyTorch + MNIST）

## 一、实验目的

1. 掌握使用 PyTorch 加载图像数据集（MNIST）并进行预处理的方法。
2. 学会设计简单的卷积神经网络（CNN）并完成训练、验证与测试。
3. 通过修改网络结构（增加卷积层、改变卷积核数量、添加 BatchNorm 与 Dropout）观察模型性能变化。
4. 掌握训练曲线绘制、模型评估及结果分析的基本技能。

## 二、实验环境

- **操作系统**：Windows / Linux / macOS
- **编程语言**：Python 3.7+
- **深度学习框架**：PyTorch 1.8+
- **其他库**：torchvision, matplotlib, numpy

## 三、数据集说明

采用 **MNIST 手写数字数据集**（Modified National Institute of Standards and Technology）。该数据集包含 0~9 共 10 个类别的手写数字灰度图像，训练集 60000 张，测试集 10000 张，每张图像尺寸为 28×28 像素。实验中将原始训练集按 8:2 随机划分为训练集（48000 张）和验证集（12000 张），测试集保持不变。

**数据预处理**：将图像转换为张量（`ToTensor`），像素值归一化到 [0,1]，并使用 MNIST 的均值（0.1307）和标准差（0.3081）进行标准化。

## 四、实验任务与流程

### 基础任务
- 加载 MNIST 数据集并划分训练、验证、测试集。
- 设计原始 CNN 模型（两层卷积 + 两层全连接）。
- 训练 5 个 epoch，记录每个 epoch 的训练/验证 loss 和 accuracy。
- 在测试集上评估模型，输出准确率。
- 绘制训练曲线（loss 曲线和 accuracy 曲线）。

### 进阶任务1：修改网络结构
**原始模型**：
- 卷积层1：输入通道1 → 32通道，核3×3，padding=1，ReLU，MaxPool(2,2)
- 卷积层2：32 → 64通道，核3×3，padding=1，ReLU，MaxPool(2,2)
- 全连接层1：64×7×7 → 128，ReLU
- 输出层：128 → 10

**改进模型**：
- 增加第三层卷积：通道数 64→128→256
- 每层卷积后加入批归一化（BatchNorm2d）
- 全连接层输出从128改为256，并添加 Dropout(0.5)
- 通过对比两组模型的测试准确率、损失曲线及收敛速度，分析网络深度与复杂度对性能的影响。

## 五、实验结论

1. **更深的网络**（增加卷积层、通道数、BatchNorm 和 Dropout）可以提升 MNIST 分类准确率，但边际收益有限，同时训练时间延长。
2. **优化器选择**对训练过程影响显著：Adam 自适应学习率，稳健高效；SGD 对学习率敏感，但最终也能达到相近精度。
3. **MNIST 数据集**较为简单，原始模型即可达到约 99% 的准确率，改进模型的优势需在更复杂的数据集（如 CIFAR-10）上才能充分体现。
4. 通过本实验，掌握了 PyTorch 模型设计、训练、验证、测试的完整流程，并学会了如何通过控制变量法对比不同网络结构和超参数的效果。
